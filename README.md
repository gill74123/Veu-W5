# Vue 直播班 Week5 - 前台產品頁面

## 主線任務說明
* 產品列表顯示
* 查看單一產品細節
* 加入購物車功能
* 購物車列表顯示
* 刪除單一、全部購物車項目
* 調整購物車產品數量
* 建立訂單

## 應用
* 產品列表相關 API
* 購物車列表相關 API
* 建立訂單相關 API
* 表單驗證套件（Vee-Validate）
* 加載效果套件（Vue Loading）
